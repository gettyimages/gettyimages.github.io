Komma igång

  1. Skapa ett fritt konto

    Registrera dig för ett kostnadsfritt konto på vår API-plats för att begära en API-nyckel

  2. Skaffa din API-nyckel

    Vilken nyckel bör jag välja?

    Testnyckel: Använd Getty Images API för att testa alla API-funktioner (bildsökning, inläsning av metadata och kontohantering) för en utvald grupp med över elva miljoner bilder. Denna nyckel erbjuder inte åtkomst till hämtningsbara bilder. Om du är ute efter hämtningsalternativ eller åtkomst till mer innehåll kan du kontakta oss via formuläret på vår webbplats.

    Inbäddningsnyckel: Använd Getty Images API för att söka genom över 84 miljoner bilder som är tillgängliga för icke-kommersiell inbäddning på din webbplats, i din app eller på sociala medier.

  3. Prova vårt API

    Använd vårt interaktiva verktyg för att utföra alla funktioner i API:et utan att skriva någon som helst kod.

  4. Ytterligare information

    Vi har dessutom SDK:er, detaljerad teknisk dokumentation och arbetsflöden med kodexempel som kan hjälpa dig att komma igång med att utveckla din app.
