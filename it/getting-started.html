Inizia subito

  1. Crea il tuo account gratuito

    Registrati per un account gratuito sul nostro sito API per richiedere una chiave API

  2. La tua chiave API

    Quale chiave dovrei scegliere?

    Test Key: usa Getty Images API per testare tutte funzionalità di API (cerca immagini, recupero di metadata e gestione degli account) per una gamma selezionata di oltre 11 milioni di immagini. Tale chiave non dà accesso alle immagini scaricabili; se sei interessato alle opzioni di download o ad accedere a maggiori contenuti, contattaci tramite il modulo sul nostro sito.

    Embed Key: usa Getty Images API per cercare tra più di 84 milioni di immagini disponibili da inserire a scopi non commerciali nel tuo sito, applicazione o social media.

  3. Prova API

    Usa il nostro strumento interattivo per eseguire tutte le funzionalità di API senza scrivere una riga di codice.

  4. Informazioni aggiuntive

    Abbiamo inoltre SDK, una documentazione tecnica dettagliata e flussi di lavoro con codice di esempio per aiutarti a creare la tua app.
