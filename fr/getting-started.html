Démarrer

  1. Créez votre compte gratuit

    Enregistrez-vous pour créer un compte gratuit sur l'API de notre site et obtenir une clé API.

  2. Obtenez votre clé API

    Quelle clé dois-je choisir ?

    Test clé : Utilisez l'API de Getty Images pour tester toutes les fonctionnalités de l'API (recherche d'images, récupération de métadonnées et gestion du compte) et accéder à une collection de plus de 11 millions d'images. Cette clé ne vous donne pas accès aux images téléchargeables ; si vous êtes intéressé(e) par les options de téléchargement ou par l'accès à davantage de contenu, veuillez nous contacter via le formulaire disponible sur notre site.

    Clé d'images embarquées : Utilisez l'API de Getty Images pour effectuer une recherche parmi plus de 84 millions d'images disponibles pour une intégration non commerciale à votre site Web, application ou réseau social.

  3. Essayez l'API

    Utilisez notre outil interactif pour exécuter toutes les fonctionnalités de l'API sans avoir à écrire de ligne de code.

  4. Informations complémentaires

    Nous disposons également de contrats de licence SDK, de documentation technique détaillée et de flux de travail avec un exemple de code pour vous aider à démarrer le développement de votre application.
