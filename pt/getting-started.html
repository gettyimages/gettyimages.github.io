Introdução

  1. Crie uma conta gratuita

    Registe-se no nosso site API para obter uma conta gratuita e solicitar uma chave da API

  2. Obtenha a sua chave da API

    Que chave devo escolher?

    Chave de teste: Utilize a API da Getty Images para testar todas as operações da API (pesquisa de imagens, extração de metadados e gestão da conta) para um conjunto selecionado de mais de 11 milhões de imagens. Esta chave não permite acesso a imagens disponíveis para download. Se estiver interessado em opções de download ou em acesso a mais conteúdos, contacte-nos através do nosso formulário disponível no website.

    Chave incorporada: Use a API da Getty Images para pesquisar mais de 84 milhões de imagens disponíveis para utilização não comercial no seu website, aplicação ou rede social.

  3. Experimente a API

    Utilize a nossa ferramenta interativa para executar todas as operações da API sem escrever uma linha de código.

  4. Dados adicionais

    Também dispomos de SDK, documentação técnica detalhada e fluxos de trabalho com código de amostra para o ajudar a começar a desenvolver a sua aplicação.
