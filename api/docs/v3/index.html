Getty Images API 3.0 Documentation

Search
GET /search/events Search for events
GET /search/images/creative Search for creative images
GET /search/images/editorial Search for editorial images
GET /search/videos Search for both creative and editorial videos
GET /search/videos/creative Search for creative videos
GET /search/videos/editorial Search for editorial videos

Images
GET /images Get metadata for multiple images by supplying multiple image ids
GET /images/id Get metadata for a single image by supplying one image id
GET /images/id/similar Search for images similar to an image

Videos
GET /videos Get metadata for multiple videos by supplying multiple video ids
GET /videos/id Get metadata for a single video by supplying one video id

Downloads
GET /downloads Returns information about a customer's downloaded assets.
POST /downloads/id Download an image
POST /downloads/images/id/post Download an image
POST /downloads/videos/id Download a video

Boards
GET /boards Get All Boards
GET /boards/board_id Get Board
PUT /boards/board_id/assets/asset_id Add Asset

Collections
GET /collections Gets collections applicable for the customer.

Countries
GET /countries Gets country codes and names.

Events
GET /events Get metadata for multiple events
GET /events/id Get metadata for a single event

Products
GET /products Get Products

Purchases
GET /purchased-images Get a list of previously purchased images

Usage
PUT /usage-batches/id Provides the api for reporting usage of assets in a batch format.

Asset Changes
PUT /asset-changes/change-sets Generates asset changes
DELETE /asset-changes/change-sets/change-set-id Confirm asset changes acknowledges receipt of asset changes.
GET /asset-changes/channels Gets the partners channel data

Asset Registration
POST /asset-registrations Registers a list of assets that a customer has stored in their system.
