Get countries list

Returns a list of country objects that contains country name, two letter ISO abbreviation and three letter ISO abbreviation.

Quickstart

You'll need an API key and access token to use this resource. Please see our Getting Started page for more information on how to sign up for an API key. There are several SDKs available for popular programming languages to help you get started.

Use the following endpoint to retrieve the list of countries used by Getty Images:

GET https://api.gettyimages.com/v3/countries

Add your API key to the request header:

Api-Key: <your api key>

Your request would look like this using curl:

curl -X GET -H "Api-Key: <your api key>" https://api.gettyimages.com/v3/countries

The response body will look like the following example:

{
    "countries": [{
        "iso_alpha_2": "AF",
        "iso_alpha_3": "AFG",
        "name": "Afghanistan"
    },
    <all of the other countries>
    {
        "iso_alpha_2": "ZM",
        "iso_alpha_3": "ZMB",
        "name": "Zambia"
    }]
}
