Getting Started

  1. Create your free account

    Register for a free account on our API site to request an API key.

  2. Get your API key

    Which key should I choose?

    Test key: Use Getty Images API to test all operations of the API (image search, metadata retrieval, and account management) for a selected collection of over 11 million images. This key does not offer access to downloadable images. If you're interested in download options or access to more content, please contact us.

    Embed key: Use Getty Images API to search over 84 million images available to embed for non-commercial use on your website, blog, or social media site.

  3. Try out the API

    Use our interactive tool to execute all of the operations of the API without writing a line of code.

  4. Additional details

    Have a look at our SDKs, detailed technical documentation, and workflows with sample code to help you get started developing your application.
