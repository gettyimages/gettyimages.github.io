Search For Creative Images

Use this endpoint to search our contemporary stock photos, illustrations and archival images by phrase, collection, event or license type. See the Filters section for more search options.

Quickstart

You'll need an api key and access token to use this resource. Please see our Getting Started page for more information. There are several SDKs available for popular programming languages to help you get started.

Use the following endpoint to search for creative images:

GET https://api.gettyimages.com:443/v3/search/images/creative?phrase=<query term>

Add your api key to the request header:

Api-Key: <your api key>

It would look like this using curl:

curl -X GET -H "Api-Key: <your api key>" https://api.gettyimages.com/v3/search/images?phrase=<query term>

The response body should look like this: 

{
  "result_count": 1,
  "images": [
    {
      "id": "<image id}",
      "asset_family": "editorial",
      "caption": "<image description>",
      "collection_code": "<collection code>",
      "collection_id": <collection id>,
      "collection_name": "<collection name>",
      "display_sizes": [
        {
          "is_watermarked": false,
          "name": "thumb",
          "uri": "<display url>"
        }
      ],
      "license_model": "rightsmanaged",
      "max_dimensions": {
        "height": 2738,
        "width": 4442
      },
      "title": "<image title>"
    }
  ]
}

Http Request Headers

Api-Key string Api-Key Required.
Accept-Language string Accept-Language parameter optional. Used to restrict the set of natural languages that are preferred as a response to the request.
Authorization string Authorization token optional. Used for applying content access restrictions. Use one of the OAuth2 grants to auto-populate this value.

Sorting and Pagination

Clients can control the subset of matching images returned in a response. This feature supports pagination scenarios. The subset, or "page" of images returned is specified by the client with an index: page, and offset: page_size.

page integer Page number.
page_size integer Size of page.
sort_order string Allows sorting of results. Possible values: best_match most_popular newest

Filters

Creative image search partitions a search into a query and an optional set of filters. At least one query parameter must be provided.

You'll need a Resource Owner Grant token to filter by product_types.

age_of_people array Allows for filtering based on age of individuals in an image Possible values: newborn baby child teenager young_adult adult adults_only mature_adult senior_adult 0-1_months 2-5_months 6-11_months 12-17_months 18-23_months 2-3_years 4-5_years 6-7_years 8-9_years 10-11_years 12-13_years 14-15_years 16-17_years 18-19_years 20-24_years 20-29_years 25-29_years 30-34_years 30-39_years 35-39_years 40-44_years 40-49_years 45-49_years 50-54_years 50-59_years 55-59_years 60-64_years 60-69_years 65-69_years 70-79_years 80-89_years 90_plus_years 100_over
artists string Free-text search, comma-separated artist search query.
collection_codes array Allows filtering by collection code.
collections_filter_type string Allows searching based on included or excluded collections. Possible values: include exclude
compositions array Allows for filtering based on image composition Possible values: abstract candid close_up copy_space cut_out full_frame full_length headshot looking_at_camera macro portrait sparse still_life three_quarter_length waist_up
embed_content_only boolean Restrict search results to embeddable images. The default is false.
ethnicity array Filters search results based on the ethnicity of individuals in an image. Possible values: black caucasian east_asian hispanic_latino japanese middle_eastern mixed_race_person multiethnic_group native_american_first_nations pacific_islander south_asian southeast_asian
exclude_nudity boolean Exclude images tagged as containing nudity. The default is false.
file_types array Identifies a file type for the image. Possible values: eps gif jpg png
graphical_styles array Identifies a graphical style for the image. Possible values: fine_art illustration photography
keyword_ids array Comma separated-list of keyword IDs. If specified, returns images containing the requested keyword(s). If keyword IDs and phrase are both specified, returns only those images matching the query phrase which also contain the requested keyword(s).
license_models array Identifies a licensing model for the image. Possible values are: royaltyfree, rightsmanaged Possible values: rightsmanaged royaltyfree
number_of_people array Allows for filtering based on the number of people in the image Possible values: none one two group
orientations array Choose images based on their aspect ratios. Possible values: Horizontal Vertical Square PanoramicHorizontal PanoramicVertical
phrase string Free-text search query.
prestige_content_only boolean Restrict search results to prestige images. The default is false.
product_types array Allows filtering by a list of product types. Possible values: easyaccess editorialsubscription imagepack premiumaccess royaltyfreesubscription

Errors

400 InvalidParameterValue Received when an invalid querystring parameter is passed or an unknown value is specified for an enumerated type.
400 UnknownProviderUri Received when an unknown provider uri is passed to entity_uris.
400 IncorrectlyFormedUri Received when an incorrectly formatted provider uri is passed to entity_uris.
400 MalformedRequest Received when a potentially malicious input string is received from the client as part of the request data.
400 InvalidPage Received if page is less than zero or greater than the number of pages in the search results.
400 InvalidPageSize Received if page_size is greater than 100.
401 AuthorizationTokenRequired Received when largest_download is added to the fields query string parameter and the Authorization header is missing or contains an invalid token.
403 UnauthorizedDisplaySize Received when mid_res_comp or high_res_comp is requested but not authorized for the given authorization token.
403 NoAccessToProductType Received when product_types is requested but not authorized for the given authorization token.
