Erste Schritte

  1. Kostenlosen Account einrichten

    Registrieren Sie sich, um Ihren kostenlosen Account auf unserer API-Site einzurichten und einen API-Schlüssel anzufordern.

  2. Ihr API-Schlüssel

    Welche Schlüssel gibt es?

    Testschlüssel: Testen Sie alle Funktionen der Getty Images API (Bildsuche, Abruf von Metadaten und Account-Management) an einer Auswahl von über 11 Millionen Bildern. Dieser Schlüssel bietet keinen Zugriff auf herunterladbare Bilder. Wenn Sie an Download-Optionen interessiert sind oder auf mehr Inhalte zugreifen wollen, dann kontaktieren Sie uns bitte über das Formular auf unserer Website

    Einbettungsschlüssel: Mit der Getty Images API finden Sie über 55 Millionen Bilder, die Sie für nicht kommerzielle Zwecke in Ihre Website, App oder Social Media-Plattformen einbetten können.

  3. API jetzt testen

    Mit unserem interaktiven Tool können Sie alle API-Funktionen nutzen - ohne selbst coden zu müssen.

  4. Weitere Informationen

    Um Sie bei der Entwicklung von Applikationen zu unterstützen, bieten wir auch SDKs, umfassende technische Dokumentation sowie Workflows mit Beispielcode an.
