Primeros pasos

  1. Crear una cuenta gratuita

    Regístrate para una cuenta gratuita en nuestro sitio API y solicita una clave para la API

  2. Obtén una clave para la API

    ¿Qué clave debo elegir?

    Clave de prueba: Utiliza la API de Getty Images para probar todas las funciones que tiene la API (búsqueda de imágenes, recuperación de metadatos y administración de la cuenta) con una selección de más de 11 millones de imágenes. Esta clave no ofrece acceso a imágenes descargables; si te interesan las opciones de descarga o quieres acceder a más contenido, usa el formulario que encontrarás en nuestro sitio para comunicarte con nosotros.

    Clave para insertar imágenes: Utiliza la API de Getty Images para hacer búsquedas en los más de 84 millones de imágenes disponibles para ser insertadas sin fines comerciales en tu sitio web, aplicación o red social.

  3. Prueba la API

    Usa nuestra herramienta interactiva para ejecutar todas las funciones de la API sin escribir una línea de código.

  4. Detalles adicionales

    También tenemos kits de desarrollo de software (SDK), documentación técnica detallada y flujos de trabajo con códigos de ejemplo para ayudarte en los primeros pasos de tu aplicación.
